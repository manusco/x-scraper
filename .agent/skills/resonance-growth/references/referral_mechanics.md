# Referral Mechanics Protocol

> **Objective**: Turn users into effective acquisition channels.

## 1. The Viral Equation
**Viral Coefficient (K)** = `Invites per User` * `Conversion Rate`
*   **K > 1**: Viral Growth (Exponential).
*   **K < 1**: Linear Amplification.

## 2. Incentive Design
*   **Double-Sided**: Give $10, Get $10. (Best for conversion).
*   **Reward Sizing**: Should be ~20% of LTV.
    *   *Formula*: `Max Reward = (LTV * Margin) - Target CAC`.

## 3. The Loop (Optimization)
1.  **Trigger**: Ask *after* a "Happy Moment" (Purchase, Milestone).
2.  **Share**: Native sharing (One-click WhatsApp/Copy Link).
3.  **Reward**: Immediate gratification (Credits > Cash).

## 4. Anti-Fraud Checklist
- [ ] **Self-Referral Block**: Device fingerprinting / IP check.
- [ ] **Minimum Activity**: Reward unlocks only after referred user *uses* the product.
- [ ] **Cap**: Max 50 referrals per user (flag for manual review).
