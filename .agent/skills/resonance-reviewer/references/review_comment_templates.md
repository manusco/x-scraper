# Review Comment Templates

> **Objective**: Standardize feedback to be constructive, actionable, and consistent.

## 1. Requesting Changes
**Template**:
```markdown
**Issue**: [Describe the problem clearly]
**Impact**: [Why this matters - Security/Performance/Maintainability]
**Suggestion**:
\`\`\`typescript
[Show improved code]
\`\`\`
```

## 2. Asking Questions
**Template**:
```markdown
**Question**: [Your question]
**Context**: I see [X], but I expected [Y].
**Goal**: [What you are trying to understand]
```

## 3. Praising Good Code
**Template**:
```markdown
**Nice!** üëè
This solution for [X] is very [elegant/performant/clean].
```
